   npm init electron-app@latest lakuboragts -- --template=webpack-typescript

 	cd lakuboragts

2. To start electron, just run

   npm start

3. To create installer, just run

   npm run make

INSTALLING ASAR
===============
npm -g install --engine-strict @electron/asar
View ASAR:
asar list app.asar

INSTALLING CHROMADB
===================
Download Python for Windows, install
pip install chromadb (make sure to run this on CMD Admmin mode)

During development, run ladb.bat on another terminal
ladb.bat
